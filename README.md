# Pesapal-social-media-analysis-
90-day social media performance analysis for a Kenyan fintech brand using Python and Excel, with an automated weekly reporting script.
# PesaPal Kenya - Social Media Performance Analysis

## The Problem
A fintech brand was posting across 4 platforms with no visibility
into which one was actually growing their audience. Equal effort
was going to every platform regardless of results.

## What I Did
- Analysed 90 days of cross-platform social media data
- Built the dataset and summary dashboard in Excel
- Ran engagement rate, platform efficiency, and content type
  analysis in Python
- Calculated the cost of distraction , hours wasted on
  underperforming platforms and followers left on the table
- Generated 4 data visualisations automatically

## What I Found
- TikTok earned 15.6 followers per hour , the highest of all platforms
- TikTok drove 57.9% of all new followers with only 25% of posts
- 40% of content creation hours were going to platforms below
  their engagement benchmarks

## The Automation
Built a Python script that regenerates the full weekly report
in under 30 seconds, charts, summary, and
recommendations included. No manual work required.
